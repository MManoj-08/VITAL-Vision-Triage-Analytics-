Suppose you're given a problem to find the first duplicate in an array of integers. Using the structured approach, you would:

3.  Break down the problem into smaller sub-problems (e.g., iterating through the array, checking for duplicates).

```python
def find_first_duplicate(arr):
    """
    Finds the first duplicate in an array of integers.

    Args:
        arr (list): A list of integers.

    Returns:
        int: The first duplicate element, or None if no duplicates are found.
    """
    seen = set()
    for num in arr:
        if num in seen:
            return num
        seen.add(num)
    return None
```
